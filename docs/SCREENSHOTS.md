# Screenshots Mockup Descriptions

1. Dashboard: sidebar navigation, KPI cards for total customers, churn rate, high-risk users, prediction volume, risk distribution doughnut chart, and latest prediction table.
2. Dataset Upload: CSV upload module with validation status, row count, and batch prediction action.
3. Manual Prediction: customer input form, churn probability meter, risk category, explanation factors, and recommended retention actions.
4. Analytics: model comparison chart showing ROC-AUC and F1-score across Logistic Regression, Decision Tree, Random Forest, and XGBoost.
5. Admin Panel: Django admin for customers, uploaded datasets, predictions, reports, recommendations, and model metrics.
6. PDF Report: generated prediction report with probability, risk level, model name, and top explanation factors.
