# Frontend

This build uses Django templates as the frontend layer for simpler deployment and viva demonstration. The UI lives in:

- `templates/`
- `static/css/app.css`
- `static/js/app.js`

The design follows a premium SaaS dashboard pattern with sidebar navigation, cards, charts, CSV upload, manual prediction, dark theme, and responsive layouts.

If you later want React, keep the DRF APIs and replace this folder with a Vite React client.
